# 進捗・工数レポート

```{warning}
本ディレクトリ配下のファイルは CI（GitHub Actions）が打刻コメントから自動生成します。
手動で編集しないでください。ローカルビルドでは最後に生成された内容、
または初期プレースホルダが表示されます。
```

このセクションでは、各 issue への打刻（`WORK_START` / `WORK_END`）を集計した
作業時間のレポートを公開します。データソースは GitHub issue のコメントであり、
`scripts/collect_worklog.py` が集計します。
